# Advanced Usage

This page covers advanced techniques for using the Zopio CLI, including environment variables, custom templates, and TypeScript integration.

## Environment Variables

The Zopio CLI respects the following environment variables to customize its behavior:

| Variable | Description | Default |
|----------|-------------|---------|
| `ZOPIO_CONFIG_PATH` | Custom path to the configuration file | `.zopio/config.json` |
| `ZOPIO_LOCALE` | Override the default locale | `en` |
| `ZOPIO_PLUGINS_DIR` | Additional directory to load plugins from | `.zopio/plugins` |
| `ZOPIO_TEMPLATES_DIR` | Custom templates directory | `.zopio/templates` |
| `ZOPIO_LOG_LEVEL` | Set the logging level | `info` |

## Custom Templates

You can create custom templates for code generation by adding them to your project's `.zopio/templates/` directory.

```
.zopio/templates/
├─ component/
│  ├─ index.ts.hbs
│  └─ styles.css.hbs
├─ page/
│  └─ index.tsx.hbs
└─ model/
   └─ schema.ts.hbs
```

Templates use Handlebars syntax for dynamic content:

```handlebars
// {{name}}.ts
import { useState } from 'react';

export function {{pascalCase name}}() {
  const [state, setState] = useState(null);
  
  return (
    <div className="{{kebabCase name}}">
      {{#if withHeading}}
      <h2>{{titleCase name}}</h2>
      {{/if}}
    </div>
  );
}
```

## TypeScript Support

The Zopio CLI is fully implemented in TypeScript and generates TypeScript code by default. You can customize the TypeScript configuration for generated code using the `.zopio/tsconfig.json` file.

```json
{
  "extends": "../tsconfig.json",
  "compilerOptions": {
    "strict": true,
    "target": "ES2020",
    "module": "ESNext",
    "moduleResolution": "node",
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true
  },
  "include": ["../src/**/*"]
}
```

## Türkçe Dil Desteği

Zopio CLI varsayılan dil olarak Türkçe'yi kullanacak şekilde yapılandırılabilir:

```json
// .zopio/config.json
{
  "defaultLocale": "tr",
  "supportedLocales": ["tr", "en", "de", "es"],
  "fallbackLocale": "en"
}
```

Bu yapılandırma ile CLI çıktıları Türkçe olarak görüntülenir:

```bash
$ zopio init
✅ Proje başarıyla oluşturuldu
📁 Dizin yapısı hazırlandı
🌐 Türkçe dil desteği etkinleştirildi
📦 Bağımlılıklar yükleniyor...
```
